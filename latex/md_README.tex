{\bfseries{A simple, pixel-\/based driving game/simulation focused on realistic drift physics.}}

{\ttfamily pixel\+Drift} is a C++ project that uses the {\bfseries{SFML}} library. The goal is to create a fun, technical drift experience by using a dynamic friction model. The model checks the angle between the car\textquotesingle{}s speed and its direction to create realistic drifting.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Features (v0.\+5.\+0 -\/ Core Mechanics Beta)}{Features (v0.5.0 - Core Mechanics Beta)}}\label{md_README_autotoc_md1}
This beta version focuses on the main driving parts of the game.


\begin{DoxyItemize}
\item {\bfseries{Dynamic Drift Physics\+:}} The friction changes based on how much the car is turning. This gives a realistic and satisfying drift feeling.
\item {\bfseries{Advanced Gear System\+:}}
\begin{DoxyItemize}
\item 7 Forward gears and Reverse (R).
\item Automatic gear (Auto) mode is available.
\item The game suggests the best gear and warns you if you are in the wrong one (by changing the gear text color).
\end{DoxyItemize}
\item {\bfseries{User Interface (UI)\+:}}
\begin{DoxyItemize}
\item Speedometer and RPM gauges.
\item A Mini-\/\+Map to show your position.
\item FPS (Frames Per Second) counter.
\end{DoxyItemize}
\item {\bfseries{Game Controls\+:}}
\begin{DoxyItemize}
\item Functional handbrake.
\item Boundary control to keep the car on the track.
\item A basic menu to start and exit the game.
\end{DoxyItemize}
\item {\bfseries{Tire Marks (Skid Marks)\+:}} Basic tire marks appear when you use the handbrake.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Setup and Build (\+C++)}{Setup and Build (C++)}}\label{md_README_autotoc_md2}
